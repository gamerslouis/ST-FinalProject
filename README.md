# ST-FinalProject

## Development
1. Install [nodejs](https://nodejs.org/en/download/)
2. Install yarn
    ```shell=
    npm install --global yarn
    ```
3. Install dependencies
    ```shell=
    yarn install
    ```
4. Develop with commands
    ```shell=
    yarn dev # run development server
    yarn format # format js files
    yarn link # eslint
    ```

5. Run jest unit test 
    ```
    yarn test
    ```
